# Bozor.tj — Доска объявлений Таджикистана 🇹🇯

Бесплатная платформа для покупки-продажи товаров, услуг и поиска работы в Таджикистане.

## 🚀 Быстрый старт

### 1. Установить Node.js
Скачай с https://nodejs.org/ (версия 18+)

### 2. Клонировать проект
```bash
git clone https://github.com/YOUR_USERNAME/bozor-tj.git
cd bozor-tj
```

### 3. Установить зависимости
```bash
npm install
```

### 4. Запустить локально
```bash
npm run dev
```
Откроется на http://localhost:3000

### 5. Собрать для production
```bash
npm run build
```

## 🎨 Возможности

✅ **Все бесплатно** — безлимит объявлений, поднятие, магазины  
✅ **Двуязычность** — Русский (RU) и Таджикский (ТЧ)  
✅ **Поиск с фильтрами** — по цене, году, марке, характеристикам  
✅ **Личный кабинет** — мои объявления, избранное, отзывы, баланс  
✅ **Чат** — прямое общение между продавцом и покупателем  
✅ **Магазины** — страница для бизнеса с рейтингом и отзывами  
✅ **Админ-панель** — управление объявлениями, пользователями, жалобами
